## Featured Project

### 🏦 Nigerian Digital Lending — Credit Risk Model
> Production ML pipeline predicting loan default probability for digital lenders

- Engineered 22 credit risk features in **PostgreSQL** simulating a
  Nigerian fintech feature store (mapped to Mono/Okra, CRC bureau,
  CBN DTI guidelines)
- Trained and tuned **Logistic Regression, Random Forest, and XGBoost**
  using scikit-learn pipelines
- Champion model: **XGBoost — AUC 0.7223**
- Stack: Python • PostgreSQL • scikit-learn • XGBoost

👉 [View Repository](https://github.com/KingsleyElo/nigerian-credit-risk)
